What is the Download Router?
The download router is here to help you keep your download folder tidy. Define how you want to organize your downloads folder, and Download Router takes care of the rest.
But how do I use it?

Download Router functions by means of "route rules". Apart from having a name (which is just a name, nothing else), a route rule consists of two main parts: A URL-matcher and a Target Directory. The URL-matcher is a regular expression that should match the url for which you want the route rule to trigger. This may be something like this: .*example\.com.*

The Target Directory setting for a route rule is simply the target for the downloaded file. A setting like "MyFolder" will simply put the files that are picked up by this route in a folder called "MyFolder".

The Target Directory may also be a bit more dynamic, using groups from the URL-matcher. If your URL-matcher looks like this: .*(example\d+).* and your TargetDirectory is $1, your files will be downloaded to a folder called "exampleX" where X is a number that matches the domainname from which you download.

What's a regular expression (regex)?
Regular expressions are common in the world of computer programming, and there's no chance in a blue moon that I can cover it here. If you don't know what a regular expression is, please take comfort in the fact that the default route provided with the extension is probably going to be enough for most of your needs. In case you still feel you have too much time on your hands, there are several good resources on the web for learning about regular expressions.
How can I arrange my downloads in directories based on date?
Since version 0.0.7, DownloadRouter has support for date-based sorting. In its simplest form, you use the format $d{xxx} in your Target Directory, where xxx is the date format you want.
Example 1: To arrange into a directory with this month's numeral value (01 for January, 08 for August, etc), use this value for your Target Directory: $d{MM}
Example 2: To arrange into a directory with this year's numeral value (four digits), use this value for your Target Directory: $d{YYYY}
Example 3: To arrange into a directory with this year's numeral value (two digits), and a sub directory for each month, use this value for your Target Directory: $d{YY}/$d{MM}
Behind the scenes, I'm using moment.js, so for more advanced formats, you can have a look at the format sections on their homepage, or shoot me an email and I'll help if I can.
What's this default route thingy?
The default route thingy is simply the most logical way to start using Download Router. It will put your content in a folder named after the domain from which you download it, but without any leading www:s. Eg, if you download a file called myfile.txt from the url http://www.example.com/some/path/here/myfile.txt, Download Router will put your file in a folder called "example.com" in your downloads folder.

Anonymous statistics! Are you spying on me?!
No! Whereas I recommend not downloading illegal/unethical stuff from the internet, it's none of my concern. The statistics contain no information about you or your downloads, only metadata about how the application is working. By ticking that box, you help me by understanding if something's not working so I can fix it. But if you feel uncomfortable with this, there are two measures you can take:
  1. Un-tick the Allow anonymous statistics box on the settings page
  2. Ask a professional what (s)he can do to help you with your paranoia
